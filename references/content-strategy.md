# Content Strategy

## Priority Order

1. AI news brief
2. YouTube summary or rewrite
3. Deep analysis
4. Tutorial
5. Tool roundup
6. Trend tracking

## Routing Rules

- Route to `ai_news_brief` when the topic is new, time-sensitive, and tied to a product launch, benchmark, funding event, or public statement.
- Route to `video_summary` when the source video itself contains the main value and needs structured Chinese retelling.
- Route to `deep_analysis` when multiple sources are needed to explain why the event matters.
- Route to `tutorial` only when the user explicitly wants step-by-step instruction.
- Route to `tool_roundup` only when the topic naturally expands into a comparison set.

## Topic Filter

Prefer topics that score well on all three dimensions:

1. Strong signal: the topic is clearly AI-related and timely.
2. Strong depth: the topic supports explanation, comparison, or interpretation.
3. Strong distribution: the title and opening can create curiosity without becoming false or vague.
