# Failure Handling

## Search Failure

- Narrow query terms.
- Reduce time range noise.
- Raise source thresholds.

## Transcript Failure

- Mark transcript quality as uncertain.
- Increase external source verification.
- Avoid over-claiming from partial captions.

## Writing Failure

- Fall back to a shorter article.
- Prioritize clarity over flourish.
- Preserve the fact sheet and review notes.

## Image Failure

- Continue article generation.
- Leave explicit image placeholders.
- Record the missing asset in review notes.

## Publish Failure

- Save the final HTML locally.
- Save image references locally.
- Return a manual upload package.
