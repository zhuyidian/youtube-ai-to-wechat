# Workflow

## Goal

Turn a topic or keyword set into a Chinese WeChat draft with strong source selection, readable formatting, and draft-box-safe output.

## End-to-End SOP

1. Normalize the task input and set defaults.
2. Expand the topic into Chinese and English search queries.
3. Search YouTube and collect a candidate pool.
4. Score and rank the pool.
5. Select one primary video and supporting sources.
6. Extract subtitles or transcripts with timestamps.
7. Build a fact sheet and a claim-verification list.
8. Add supplemental research from official or primary sources.
9. Route the article type.
10. Build an outline.
11. Write the information pass.
12. Rewrite in WeChat style.
13. Generate titles, summary, and cover copy.
14. Plan image slots.
15. Generate cover, inline images, and infographic prompts.
16. Format the final HTML.
17. Inject fixed branding blocks.
18. Export a review package.
19. Upload to WeChat draft box.
20. Stop and wait for human review.

## Output Package

Ship these artifacts for every run:

1. `task.json`
2. `manifest.json`
3. `sources.json`
4. `outline.md`
5. `article.md`
6. `article.html`
7. `titles.json`
8. `images/`
9. `review-notes.md`
